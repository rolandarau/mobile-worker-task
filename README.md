# TimesheetWidget
This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 11.0.3.

## Before starting
Go to `timesheet-widget` directory <br/>
Install dependencies via `npm install`

## Starting application
Run `npm start` for to start application and json-server. Navigate to `http://localhost:4200/`.

*In case of json-server won't start, use `npm start:frontend` and `npm start:server` commands.*

## Running unit tests

Run `npm run test:coverage` to execute the unit tests with coverage report.
